4-Tier Web App Generator Help Page

This application is used to design 4-tier web applications

Here are some further details regarding this applications functions:
- Each element added to the page can be done by clicking its corresponding button in the table.
- When you want to view you window, press the Open Window button at the button of the screen.
- Some of the elements can be dragged and drop on the new page (indicated in Drag and Drop? column in table).
- The database input box allows users to indicate the name of the database they are trying to reach.
- The diplay, new item, update item, and delete item all access the specified database and if none of these options are present you cannot access the database. ONE of these options must be included to access the database. If the submit button is clicked then the application will error. You can also only have one of each of the database commands on your page. The buttons will be disabled until the page is reset.
- The input item form identicates which data you want to edit in the database.
- The submit and reset buttons can be dragged and dropped by their borders to prevent click them.
- The reset window option resets the contents of the window.
- The close page option closes the window.

Format of input files: Files can be uploaded via the upload file functionality. As of now, input files must be three columns and consist of integers. This should be in a csv file.

Database set up: Inorder to set up the database use the file prjdb.sql to initialize the database. This can be found in the prj foler or with localhost/isp/prj/prjdb.sql. To initialize, run this query in your mysql database (which should be named isp).
